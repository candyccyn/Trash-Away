# Trash Away Project

Trash Away is the automatic trash that you can control to wherever you want by using your mobile phone.

# Tools 
- Raspberry Pi Pico </br>
- TCRT5000 Infrared Sensor </br>
- Micro Servo motor (SG90) </br>
- LCD Display</br>
- HC-06 Bluetooth module</br>
- 18650 Battery</br>
- Maker Drive</br>
